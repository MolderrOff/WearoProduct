# Backend для интернет-магазина

Разработан базовый функционал серверной части интернет-магазина с соблюдением принципов чистой архитектуры.

## Технологические особенности
*   **Clean Architecture**: Четкое разделение ответственности между слоями приложения.
*   **REST API**: Реализован стандартный набор CRUD-операций.
*   **Dependency Injection (DI)**: Использовано внедрение зависимостей для обеспечения гибкости и тестируемости кода.
*   **Validation**: Настроена базовая валидация входящих данных.

## Настройка и запуск

1.  **Конфигурация**: 
    *   Файл решения `.sln` находится в папке `WearoSkillbox`. 
    *   Основной проект — `.Host`. 
    *   В файле `appsettings.json` замените строку `SqlConnection` на актуальную для вашего сервера.
2.  **База данных**: 
    *   Проект настроен на работу с **MS SQL**. 
    *   Если вы используете другую СУБД, внесите соответствующие изменения в файл `InfrastructureConfigurator.cs`.

## Миграции и обновление БД
Для создания и обновления структуры базы данных выполните в терминале следующие команды:

```bash
# Установка инструментов Entity Framework (если не установлены)
dotnet tool install --global dotnet-ef

# Создание первой миграции
dotnet ef migrations add InitialCreate

# Применение миграций к базе данных
dotnet ef database update
